# Paradise Nursery Shopping App

## Project Overview

Paradise Nursery Shopping App is a React-based e-commerce application that allows users to browse a variety of plants, add them to a shopping cart, update quantities, remove items, and view the total cost before checkout.

## Features

- Landing page with Welcome message
- Product listing page
- Plants grouped by categories
- Add plants to cart
- Shopping cart with quantity management
- Remove items from cart
- Display total number of items
- Display total cost
- Continue Shopping button
- Checkout button

## Technologies Used

- React
- Redux Toolkit
- React Router
- JavaScript
- HTML
- CSS

## Getting Started

1. Clone the repository.

```bash
git clone https://github.com/Darshini2125/paradise-nursery-app.git
```

2. Navigate to the project folder.

```bash
cd paradise-nursery-app
```

3. Install dependencies.

```bash
npm install
```

4. Start the development server.

```bash
npm start
```
